# 📖 About

This repository contains implementations of the **Queue** data structure in **C**. It includes different queue implementations, core operations, and practical applications to help students understand the FIFO (First In First Out) principle.

The repository is designed for students learning **Data Structures & Algorithms**, preparing for coding interviews, or practicing university assignments.
